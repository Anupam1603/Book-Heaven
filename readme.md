# BookHeaven
## Introduction
- This project is a full-featured bookstore application built using the MERN stack (MongoDB, Express, React, Node.js). It includes user and admin roles, authentication and authorization, shopping cart functionality, and the ability to create orders.

## Features
- User
- User registration and login
- Browse books
- Add books to cart
- Create orders
- View order history
- Admin
- Admin login
- Manage books (add, update, delete)
- View all orders
- Manage users
- Common
- JWT authentication and authorization
- Responsive design
### Technologies Used
- Frontend: React, Redux, Bootstrap
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT (JSON Web Tokens)
- State Management: Redux
- Other Libraries: Axios, Mongoose, bcryptjs
- Installation
- Prerequisites
- Node.js
- MongoDB

## Installation
#### Prerequisites
- Node.js
- MongoDB

### Setup environment variables:
Create a .env file in the backend folder and add the following:
```
NODE_ENV=development
PORT=5000
MONGO_URI=your_mongo_connection_string
JWT_SECRET=your_jwt_secret
```
